-   geradores são funções que nos fornecem um iterador
-   objetos geradores podem ser pausados e retornados um valor usando a palavra-chave yield
-   chamar next() retoma a função e obtemos valores sucessivos
-   geradores assíncronos combinam o poder de async/await e geradores
-   geradores assíncronos retornam um objeto gerador assíncrono, que é um iterador
-   geradores são excelentes para iteração controlada, como ao implementar paginação
